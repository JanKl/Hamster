# Hamster
Logik für den Hamster aus dem <a href="http://www.java-hamster-modell.de/">Java Hamster Modell</a>. Entstanden im dritten Semester Angewandte Informatik an DHBW Stuttgart Campus Horb.

Ziel war es, den Hamster so zu programmieren, dass er möglichst viele Körner in einem Raum aufsammelt. Vorbild in der Vorlesung war dabei ein Staubsaugerroboter.
